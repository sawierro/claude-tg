# Установка Claude-TG

Телеграм-бот для удалённого управления Claude Code с телефона. Подключаешься к сессии Claude в терминале и управляешь ей из Telegram: видишь ответы, отправляешь команды, синхронизируешь контекст.

---

## Что нужно

- Python 3.11 или новее
- Claude Code CLI (установлен и залогинен)
- Telegram

---

## Шаг 1. Создай бота в Telegram

1. Открой Telegram, найди @BotFather
2. Отправь `/newbot`
3. Придумай имя (любое) и username (должен заканчиваться на `bot`)
4. BotFather выдаст токен вида `7123456789:AAF...` — скопируй его

---

## Шаг 2. Скачай и установи

**Windows:**
```
git clone https://github.com/sawierro/claude-tg.git
cd claude-tg
setup.bat
```

**Linux / Mac:**
```
git clone https://github.com/sawierro/claude-tg.git
cd claude-tg
chmod +x setup.sh
./setup.sh
```

Скрипт спросит токен бота — вставь тот что от BotFather. Всё остальное сделает сам.

---

## Шаг 3. Первый запуск

**Windows:** `start.bat`
**Linux / Mac:** `./start.sh`

В Telegram найди своего бота и отправь `/start`. Бот запомнит тебя — теперь только ты можешь им управлять.

---

## Как пользоваться

### Основной сценарий

1. Запусти `claude` в терминале как обычно, поставь задачу
2. В Telegram отправь `/connect`
3. Появятся кнопки с активными сессиями — нажми нужную
4. Готово:
   - Ответы Claude из терминала приходят в Telegram
   - Пишешь сообщение в чат — оно уходит в сессию Claude
   - Claude выполняет команду и присылает результат

### Синхронизация контекста

Когда поработал через Telegram и возвращаешься к терминалу:

1. В Telegram: `/sync`
2. Бот создаст файл `.claude-tg-sync.md` в папке проекта
3. В терминале скажи Claude: "прочитай .claude-tg-sync.md"

Claude в терминале получит полный контекст того, что было сделано через Telegram.

### Команды

| Команда | Что делает |
|---------|------------|
| `/connect` | Показать сессии из терминалов, подключиться |
| `/sync` | Записать сводку задач для терминала |
| `/new имя . задача` | Создать новую сессию |
| `/sessions` | Список подключённых сессий |
| `/stop имя` | Отключить сессию |
| `/help` | Справка |

---

## Повседневный запуск

После первой настройки всё просто:

1. Запускаешь `start.bat` (Windows) или `./start.sh` (Linux/Mac)
2. В Telegram: `/connect` → выбираешь сессию
3. Работаешь

Бот нужно запускать на том же компьютере, где работает Claude Code.

---

## Если что-то не работает

**"Claude CLI not found"** — убедись что `claude` доступен в терминале. Попробуй `claude --version`.

**Бот не отвечает** — проверь что `start.bat` запущен и не закрыт.

**"Not logged in"** — выполни `claude login` в терминале.

**Нет сессий в /connect** — сначала запусти `claude` в отдельном терминале.
